# 🌸 Mother’s Day Interactive Website

An interactive and animated frontend experience created as a Mother’s Day surprise for my mum.

This project combines storytelling, animations, music, and responsive design into a personalized digital experience that can be fully customized with your own images, audio, and messages.

## ✨ Features

* Password-protected entrance
* Smooth GSAP animations and transitions
* Responsive layout for desktop and mobile
* Interactive image gallery
* Music player integration
* Animated digital letter
* Soft pastel glassmorphism UI
* Fully customizable content

## 🛠️ Technologies Used

* HTML5
* CSS3
* JavaScript
* GSAP Animation Library
* GitHub Pages

## 📱 Responsive Design

This project was designed to adapt across:

* Desktop
* Tablets
* Mobile devices

Responsive typography, spacing, and layouts were implemented using:

* Flexbox
* CSS clamp()
* Viewport-based sizing
* Mobile-friendly interactions

## 💡 Project Purpose

The goal of this project was to create a meaningful digital experience while improving frontend development skills and experimenting with AI-assisted workflows.

The entire application was planned, developed, debugged, and deployed in only a few hours using AI as a pair-programming and troubleshooting assistant.
